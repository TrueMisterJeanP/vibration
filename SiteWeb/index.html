Edition Community

Une base publique, utile et vérifiable.

La version Community reste sous GPL sur GitHub. Elle permet de tester, auditer et utiliser le socle de messagerie chiffrée sans dépendre d'un service centralisé.

Socle commun

Conversations privées, groupes, fichiers, appels, tableau blanc et interface PWA partagent le même code de base.

Licence GPL

Le code publié peut être étudié, modifié et redistribué dans les conditions de la GPL-3.0-or-later.

Chemin Enterprise

Les corrections communes sont appliquées au socle, puis la version Community est régénérée depuis le projet maintenu.

Edition Enterprise

Une offre self-hosted pour reprendre le contrôle.

L'entreprise reçoit une version complète sous GPL, installée sur son infrastructure, avec accompagnement serveur, Coturn, base de données, sauvegardes et support.

Voir l'offre Enterprise
Serveur Go maîtriséDéploiement sur serveur dédié, VPS, réseau interne ou infrastructure client.
Coturn privéAppels audio et vidéo plus fiables hors réseau local, sans dépendre d'un relais tiers.
Console d'administrationRôles, révocation de sessions, bannissement, journal, configuration WebRTC et base de données.
Support opérationnelInstallation, maintenance, mises à jour, sauvegardes, restauration et durcissement HTTPS.

Souveraineté technique

L'organisation garde la main sur l'infrastructure.

  • Serveur Go, données et fichiers administrés par l'entreprise ou son prestataire.
  • Coturn privé pour contrôler le relais des appels WebRTC.
  • Messages, noms de fichiers et types MIME chiffrés avant envoi au serveur.
  • Support Enterprise vendu comme service, sans retirer les droits accordés par la GPL.

Déploiement

Un serveur Go, une interface web, un Coturn privé.

Community permet d'évaluer le socle. Enterprise ajoute l'accompagnement nécessaire pour un déploiement maîtrisé en production.
